

## Install packages

```bash
pip install virtualenv
```

### Paquetes necesarios para SQL y login
```bash
    pip install flask-WTF
    pip install flask-mysqldb
    pip install Flask
    pip install Flask-Login
```

## Crear entorno virtual

```bash
    python -m venv env
```

### Activar Entorno Virtual

```bash
    .\env\Scripts\activate
```

## Correr app.py
```bash
    python .\src\app.py
```
